## Overview

AMACS is a minimalist yet powerful code editor built from scratch in C, leveraging GPU acceleration for smooth rendering performance.

## Key Features

### GPU-Accelerated Rendering
Utilizes SDL2 and OpenGL for hardware-accelerated text rendering, ensuring smooth scrolling and editing even with large files.

### Custom Lexer
Built a custom lexer for C/C++ that provides:
- Real-time syntax highlighting
- Cursor tracking
- Code selection and manipulation

### Productivity Features
- Regex-based search across files
- File system manipulation
- Multiple cursor support
- Keyboard-driven workflow

## Technical Implementation

The editor is built with a focus on:
- Low memory footprint
- Fast startup time
- Responsive UI even with large codebases
- Extensible architecture for adding language support
