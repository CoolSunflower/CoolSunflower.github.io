## Overview

ZoionNet is an acoustic wildlife detection system designed to mitigate human-animal conflict in biodiversity hotspots like Methagiri.

## Recognition

**Honorary Mention** at IEEE COMSOC 2024 for innovative approach to conservation technology.

## Key Features

### Dual-State CNN Classifier
Built on embedded systems, achieving 94.5% accuracy in species identification using acoustic signatures.

### Real-Time Alert System
Integrated alerts notify forest rangers and authorities when potentially dangerous wildlife is detected near human settlements.

### Cloud Dashboard
Long-term data analysis platform for tracking wildlife patterns and movement over time.

## Impact

- Cost-effective, scalable solution
- Cellular connectivity for remote deployment
- Helps protect both wildlife and local communities
