## Overview

Hospital Management Information System (HMIS) is a comprehensive healthcare platform designed to streamline hospital operations, from patient management to billing and analytics.

## Key Features

### Admin Analytics Panel
Real-time insights on patient trends, financial metrics, and staff performance for data-driven decisions.

### Recurring Email Notifications
Implemented a scalable notification system with BullMQ + Redis, powering automated reminders, payroll updates, and system alerts.

### Multi-Role Authentication
Secure authentication system supporting different user roles: Admin, Doctor, Nurse, Receptionist, and Patient.

## Technical Highlights

- **Scalable Architecture**: Built with microservices principles, enabling horizontal scaling
- **Real-time Updates**: WebSocket integration for live data synchronization
- **Comprehensive API**: RESTful API with full CRUD operations
- **Production Ready**: Deployed with 60-member team collaboration
