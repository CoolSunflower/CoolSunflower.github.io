## Overview

A complete compiler front-end implementation for the MicroC language, demonstrating compiler construction principles.

## Features

### Lexical Analysis (Flex)
- Token recognition for keywords, identifiers, literals
- Error reporting with line numbers
- Comment handling

### Parsing (Bison)
- Context-free grammar implementation
- Abstract Syntax Tree (AST) generation
- Semantic analysis

### Symbol Table
- Scoped symbol management
- Type checking
- Function and variable declarations

### Code Generation
- Three-Address Code (TAC) generation
- Quadruple representation
- Enables low-level program analysis

## Supported Language Features

- Data types: int, float, char
- Control flow: if-else, while, for
- Functions with parameters
- Pointers and arrays
- Arithmetic and logical operators
